# Kanji Benkyou

This is a project for kanji studies.
It uses the data from [kanjiapi.dev](https://kanjiapi.dev/)

## Features

General:
- [x] Searching / filtering
- [x] PDF sheet generation
- [ ] Translations

Per user:
- [ ] Flashcards/Decks
- [ ] Keep track
- [ ] Review
- [ ] Quiz 

## Stack
- Django + DRF 
- PostgreSQL for users
- Vue3 + Ant Vue Design
