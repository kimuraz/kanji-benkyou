# Kanji Benkyou

This is a project for kanji studies.
It uses the data from [kanjiapi.dev](https://kanjiapi.dev/)

## Features

General:
- [ ] Searching / filtering
- [ ] PDF sheet generation

Per user:
- [ ] Bookmark (Flashcards)
- [ ] Keep track
- [ ] Review
- [ ] Quiz 

## Stack
- [ ] Django + DRF 
- [ ] PostgreSQL for users
- [ ] Elasticsearch for kanji data
- [ ] Vue3
- [ ] TailwindCSS for styling
